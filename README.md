## Running perfomance Tests

To run perfomance tests, run the following command

```bash
  git clone https://github.com/MaheshMadushan/ParallelLinkedListPerformance.git
  cd ParallelLinkedListPerformance/performance/build/
  ./perf <num of threads>
```
